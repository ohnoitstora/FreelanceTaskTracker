# Freelance Task Tracker

A terribly coded Textual TUI app for tracking freelance tasks. 

## Features
- Add tasks via input and button
- See tasks in a list
- Sync button (freezes UI for 2 seconds)
- All state in a global variable
- All code in one file
- No async, no custom widgets, no external CSS

## Usage

```bash
pip install -r requirements.txt
python app.py
```
